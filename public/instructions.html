2 The Timer Banner ▼
2.1   Starting and Stopping the Timer

Once your session loads, the dark banner at the top contains all timer controls. The circular button in the centre toggles between Start and Stop.

00:00:00

  • Click Start to begin counting. The button label changes to Stop.
  • Click Stop to pause. The timer display becomes editable (see 2.2).
  • Click Start again to resume from exactly where it paused.
2.2   Editing the Time

If the timer drifts out of sync with your show, you can correct it without losing any stamps.

+ Stamp
01:23:45
Start
Save
  1. Click Stop — the timer becomes an editable text field pre-filled with the current time.
  2. Type the correct time in HH:MM:SS format.
  3. Press Enter to confirm, or Escape to cancel without changing anything.
  4. Click Start to resume from your corrected time.
3 Creating Timestamps ▼
3.1   Creating a Stamp

When something worth noting happens in the show, capture it instantly using either method:

  • Click the + Stamp button on the left side of the banner.
  • Press the * (asterisk) key on your keyboard as a quick shortcut.

A new card appears at the correct position in the list, sorted by its captured time.

3.2   Entering a Topic

Each stamp card shows a time badge and a topic input. The input is automatically focused after stamping so you can type right away.

4 Managing Your Log ▼
4.1   Creating Subtopics

Timestamps can be nested under one another to represent topics and their sub-discussions.

  1. Click and hold a stamp card to begin dragging it.
  2. Drag it on top of the card you want as its parent.
  3. Release — the dragged card becomes a subtopic indented beneath the parent.

This is useful when an episode has a main topic with sub-points discussed at different times.

4.2   Adding Multiple Times to One Stamp

If a topic is revisited later in the show, attach the new time to the existing stamp rather than creating a duplicate.

  1. While the timer is running, double-click the time badge on an existing stamp card.
  2. The current time is appended to that card's time badge.

5 Timing Modes ▼
5.1   Manual Timer

The default mode. A classic start/stop timer driven by the browser's interval.

  • Best when you have no external video reference.
  • Click Start when your show begins and Stop to pause.
  • Note: Browsers throttle background tabs, which may cause slight drift over long sessions if you switch away from this tab.
5.2   Wall Clock

An improved timer that reads the system clock (Date.now()) instead of counting intervals — so it never drifts.

  • Stays accurate even when the tab is in the background or the window is minimised.
  • Behaves identically to Manual Timer from the user's perspective.
  • Recommended for long sessions or when you frequently switch tabs.
5.3   YouTube Stream

Embeds a YouTube video or live stream and drives the timer directly from the player's playback position.

  1. Select YouTube Stream on the landing page.
  2. Paste the YouTube URL or video ID into the field that appears.
  3. Submit — the player loads below the banner. Use Start to play and Stop to pause.
Live streams & offset: For live streams the timer may start at an unexpected value. Use the time editor (Stop → edit → Enter) to set the correct broadcast start time. The app handles any negative player values automatically.
  • Scrubbing the video moves the timer — stamps are still sorted correctly.
  • Resuming after a pause recalculates the offset so time continues smoothly from where it stopped.
  • Use the ▲ Hide stream toggle to collapse the player without stopping playback.
